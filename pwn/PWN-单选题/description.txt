```c
#include <stdio.h>
#include <string.h>

int main() {
    char password[10];
    printf("Enter your password: ");
    scanf("%s", password); 
    
    printf("Password saved.\n");
    return 0;
}
```

A. 12345678
B. "admin"
C. "SuperSecretPassword123"
D. 以上答案都不会导致该错误。
